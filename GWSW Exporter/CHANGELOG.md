## [0.3.0] - 2025-12-24
### Added
- Export of manholes, conduits, pressure pipes, and inlets.
- Progress is now reported in the view.
- Option to show or hide errors in the view.
- SQL query is now automatically read from a text file.

## [0.2.0] - 2025-12-19
### Added
- SQL-based data retrieval.
- Initial framework for parsing data and mapping directories to GWSW values.
- Core support for generating and saving TTL files.
- CSV export for the list overview.
- Load and save functionality for SQL queries as plain text files.
